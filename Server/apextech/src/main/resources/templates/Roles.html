Admin

Full control over the system.

  • Managge all products
  • View and delete users
  • Change user roles
  • Access admin only pages

Staff

Operational access for catalog updates.

  • Add new products
  • Browse products
  • Cannot delete users
  • Cannot change roles

Customer

Only shopping access.

  • Browse and search products
  • Register and log in
  • Cannot add or edit products
  • Cannot access admin tools
